# Lyceum Texts

Data repository for all Lyceum reader texts.

## Structure

Each text is stored in its own directory named by text slug:

```
lyceum-texts/
├── text-slug-1/
├── text-slug-2/
└── ...
```

## Generation

All content is generated by the [lyceum-pipeline](https://github.com/yourusername/lyceum-pipeline) tool.

## Usage

Set the `LYCEUM_TEXTS_DIR` environment variable to point to this directory when working with Lyceum tools:

```bash
export LYCEUM_TEXTS_DIR=/path/to/lyceum-texts
```

Or use relative paths from your project root.
